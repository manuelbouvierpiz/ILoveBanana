    class Grille

    Classe Grille :¶ ↑

    - connaît son id, sa matrice départ, sa matrice courante, sa matrice correcte et son nombre minimum de clics pour arriver à résolution
- sait dire si elle correcte, jouer sur une Case, donner sa difficulté, donner sa taille, donner une aide

    Attributes

    idGrille[R]

    • Variable d'instance accessible en lecture représentant l'unique ID de la Grille

    matrice[R]

    • Variable d'instance accessible en lecture représentant le tableau de Cases à deux dimensions de la Grille

    matriceDepart[R]

    • Variable d'instance accessible en lecture représentant le tableau de Cases à deux dimensions lorsque la Grille vient d'être créée

    nbClicMin[R]

    • Variable d'instance (un Integer) représentant le nombre minimum de clics pour remplir correctement la Grille

    # File Grille.rb, line 90
def jouer(unX, unY, rougeOuBleu)
        case rougeOuBleu
                when 'b', 'bleu'
                        @matrice[unX][unY].setBleu
                when 'r', 'rouge'
                        @matrice[unX][unY].setRouge
                when 'v', 'vide'
                        @matrice[unX][unY].setVide
        end
        return self
end
